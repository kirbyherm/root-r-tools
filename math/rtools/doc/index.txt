This directory contains the ROOT::Math::RMinimizer class
BEGIN_HTML 
The RMinimizer class is used to implement a minimizer which utilizes the R/ROOT Interface. 
 
<p>
The ROOT::Math:RMinimizer class implements the
ROOT::Math::Minimizer interface, as a new plugin with name "RMinimizer". 
The class can be used when fitting histograms or graph by selecting it
as default minimizer. This can be done via: 
<pre>
ROOT::Math::MinimizerOptions::SetDefaultMinimizer("RMinimizer");
</pre>
Via the MinimizerOptions, one can control the possible options for the
class. These can be found on the ROOT::Math::MinimizerOptions header file. 
Additionally, options for the R function optim can be found on the optim man page.
A list of the default options can be obtained by doing: 
<pre>
ROOT::Math::Minimizer * min =ROOT::Math::Factory::CreateMinimizer("Minimizer"); 
min->Options().Print(); 
<pre>


END_HTML
